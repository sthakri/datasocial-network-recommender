# Social Network Recommender

A practice project for building friend and page recommendations based on social connections.

## What it does

- Recommends friends based on mutual connections
- Suggests pages your friends like
- Cleans messy social network data

## Files

- `01_Maintaining.ipynb` - Load and explore data
- `02_Cleaning.ipynb` - Clean dirty data
- `03_people_you_mayknow.ipynb` - Recommendation algorithms
- `data.json`, `data2.json`, `massive_data.json` - Sample datasets

## How to use

Just run the notebooks in order (1 → 2 → 3). No extra packages needed, just Python and Jupyter.

## What I learned

- How recommendation algorithms work
- Data cleaning with Python
- Working with JSON files
- Using sets and dictionaries
